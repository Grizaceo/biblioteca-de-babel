# La Biblioteca de Babel — Three.js

Exploración 3D en primera persona de *La Biblioteca de Babel* de Jorge Luis Borges.

Galeria hexagonal infinita con pozo central, estantes con libros detallados, lámparas de aceite con parpadeo, escaleras funcionales entre pisos, y niebla para la profundidad.

## Requisitos

- Node.js >= 18
- navegador moderno con WebGL

## Instalar y ejecutar

```bash
npm install
npm run dev
```

Abrir `http://localhost:5173`. Click para tomar el control (Pointer Lock).
WASD para moverte, Shift para correr, Espacio para saltar.
Camina sobre los círculos en el piso de los vestíbulos para subir/bajar de piso.

## Build producción

```bash
npm run build
npm run preview
```
